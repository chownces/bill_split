## Contributions
Any code contributions to improve the application are welcome!

### Getting Started
1. Clone the repository locally.
1. `cd` to the repository directory locally.
1. Run `yarn start` to start the app on `localhost:3000`. Navigate to `localhost:3000` on your browser.

### Pull Requests
1. Before making a new commit, run `yarn run format` to format the code.
1. Make a new branch and commit atop the new branch. Please name the new branch after a shortname of the feature you are implementing.
1. Make a pull request to `master` branch on GitHub.

### Reporting an Issue
1. Please feel free to report bugs by creating a new issue on the GitHub repository.
